My first pet project made with the help of AI

  • Mar 15, 2026
  • Categories: Blog

In February I've started a pet project. My goal here is to create something that is hosted online, using AI IDE (Cursor in my case) and in the meantime learn how to use AI effectively - both as a coding assistant, and a teacher.

As a coding assistant, Cursor IDE greatly helped me to create the first HTML/CSS version of my daily tracker, with vanilla JS only.

I'm using this tracker on my phone and host it as a static index.html file here - this is my VPS on Digital Ocean.

This tracker is intended just for personal use, so I've implemented quite a limited set of features - only those that are needed for me.

Total time spent on this basic 1.0 version: 4 hours. I've also started a GitHub repo for the project code.

As a teaching assistant, Cursor and other LLMs will help me re-implement this same set of features but with different underlying frameworks/libraries - as a React app, a Vue/Svelte/Preact/Alpine app, a Django or Laravel project, and so on. Each time I will implement the same tracking tool, and my goal will be to learn a new framework, to master it at least on a beginner level.

I predict that I will have constant temptations to skip the actual learning process and just delegate coding tasks to Cursor IDE. My goal there is to figure out how I can learn new frameworks and technologies effectively with the help of LLMs as teachers/debuggers/reviewers.

This post is written without any LLM assistance.
